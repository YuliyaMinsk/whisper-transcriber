# Audio Files

This folder contains audio files for transcription.

## Supported formats:
- `.ogg` - Ogg Vorbis audio files
- `.opus` - Opus audio files  
- `.mp3` - MP3 audio files
- `.wav` - WAV audio files
- `.m4a` - M4A audio files
- `.mp4` - MP4 video files (audio will be extracted)

Place your audio files here and run the transcription script from the project root:

```bash
source whisper-env/bin/activate
python transcribe.py
```
